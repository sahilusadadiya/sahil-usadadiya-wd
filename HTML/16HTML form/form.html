Form

form tag will helps to send client data to the server

There are two methods for sending data to the server

name attribute is helps to send data

  1. GET
  2. POST

GET

will use URL for sending data to the server thats why get method is less secure

limitation 2048 char only

not supported file uploading

we can bookmark get data with URL

get will reload page with prev data

faster than post 90% searching will be done by get methods

? => query string

& => multiple data

POST

will not using URL for sending data to the server thats why get method is secure

no limitation

supported file uploading

we can't bookmark post data with URL

get will not able to do reload page with prev data

slower than get
